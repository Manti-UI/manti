# Listbox

A selectable list backed by the Zag.js listbox machine. Pass an `items` array and
the machine owns roving focus, typeahead, and the selection state.

<Demo name="listbox/basic" center />

## Basic

Each item is `{ value, label, disabled? }` — `value` keys the option, `label` is
shown, and `disabled` skips it during keyboard navigation.

<Demo name="listbox/basic" center />

## Multiple selection

Set `selectionMode="multiple"` to let several options be selected at once. Pass an
array to `value` or `defaultValue`.
