# Splitter

A resizable split layout backed by the Zag.js splitter machine. Pass a `panels`
array and drag the dividers; the machine owns the resize math, keyboard support,
and min/max constraints.

<Demo name="splitter/basic" />

## Three panels

Add more entries to `panels` and matching `defaultSize` percentages for a
multi-pane layout. Drag either divider to redistribute space.
