# DatePicker

A calendar date picker backed by the Zag.js date-picker machine. Values are ISO
date strings (`YYYY-MM-DD`); the machine owns the calendar grid, month
navigation, and keyboard selection.

<Demo name="date-picker/basic" center />

## Basic

`selectionMode="single"` picks one day. Seed it with `defaultValue` as an array
of ISO date strings.

<Demo name="date-picker/basic" center />

## Range

`selectionMode="range"` selects a start and end day. `defaultValue` takes two ISO
date strings.

<Demo name="date-picker/range" center />
