# Select

A single or multi select backed by the Zag.js select machine. The machine owns
keyboard navigation, typeahead, and dismissal; you supply an `items` array and
it renders the trigger plus a portalled translucent listbox.

<Demo name="select/basic" center />

## Basic

Each item is `{ value, label, disabled? }`. The trigger shows the `placeholder`
until something is selected.

<Demo name="select/basic" center />

## Multiple

Set `multiple` to allow more than one selection; the trigger summarizes the
chosen values.
