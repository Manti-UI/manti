# Menu

A dropdown command menu backed by the Zag.js menu machine. The machine owns
keyboard navigation, typeahead, and dismissal; you supply the trigger and the
`items`.

<Demo name="menu/basic" center />

## Basic

`trigger` is cloned with the machine's trigger props, so it works with any
interactive element. Each command is `{ value, label, icon?, shortcut?, disabled? }`,
and `onSelect` fires with the selected command's `value`.

<Demo name="menu/basic" center />

## Groups and separators

Mix `{ type: 'group', label, items }` and `{ type: 'separator' }` entries into the
`items` array to cluster commands under headings.
