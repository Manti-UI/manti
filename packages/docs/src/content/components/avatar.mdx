# Avatar

An avatar with graceful image-load fallback, backed by the Zag.js avatar machine.
It shows the image when it loads and falls back to initials otherwise.

<Demo name="avatar/basic" center />

## Sizes

`size` ranges from `sm` to `xl`. The image is clipped to the avatar's shape.

<Demo name="avatar/basic" center />

## Fallback and shape

Omit `src` (or let it fail) and the `children` render as a fallback. Switch
`shape` between `circle` and `square`.
