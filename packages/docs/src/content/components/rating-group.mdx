# RatingGroup

A star rating backed by the Zag.js rating-group machine. The machine owns
keyboard control, hover preview, and form wiring; you set the icon count and
precision.

<Demo name="rating-group/basic" center />

## Half stars

Set `allowHalf` to permit half-star precision — the fill clips to the hovered or
selected half.
