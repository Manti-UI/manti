# TagsInput

A multi-value entry field backed by the Zag.js tags-input machine. Type and press
Enter to add a tag, Backspace to remove the last one; the machine owns editing,
navigation, and the hidden form value.

<Demo name="tags-input/basic" />

## States

Field flags — `invalid`, `disabled`, `readOnly` — and `max` to cap the number of
tags behave like any Manti form control.
