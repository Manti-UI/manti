# Steps

A step-by-step flow backed by the Zag.js steps machine. Pass an `items` array of
`{ title, description, content }`; the machine tracks the active step and the
built-in controls advance it.

<Demo name="steps/basic" />

## Vertical

Set `orientation="vertical"` to stack the steps as a left rail with the active
panel beside it.
