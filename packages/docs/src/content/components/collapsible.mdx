# Collapsible

A single open/close disclosure backed by the Zag.js collapsible machine. The
content animates its height smoothly as it opens and closes.

<Demo name="collapsible/basic" center />

## Basic

Pass the toggle label as `trigger` and the revealed content as `children`. Use
`defaultOpen` to start expanded, or `open` with `onOpenChange` to control it.
