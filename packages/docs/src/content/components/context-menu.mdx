# ContextMenu

A right-click (or long-press) context menu backed by the same Zag.js menu machine
as Menu, opened at the pointer. Wrap any region in `children` and pass `items` —
commands, separators, and titled groups. It reuses the `menu` style scope, so its
panel matches the dropdown menu.
