# NumberInput

A numeric stepper backed by the Zag.js number-input machine. The machine owns
clamping, keyboard stepping, and the increment/decrement triggers; you set the
range and step.

<Demo name="number-input/basic" center />

## Basic

`min`, `max`, and `step` bound and quantize the value. The value mirrors the
input as a string, and `onValueChange` also receives it parsed as a number.

<Demo name="number-input/basic" center />

## Sizes

`size` sets the control height: `sm`, `md` (default), or `lg`.
