# Components

Manti UI is a framework-agnostic design system built on [Zag.js](https://zagjs.com)
behavior machines. Each component pairs a headless state machine — which owns
keyboard handling, focus, forms, and ARIA — with token-driven CSS keyed to the
public `data-scope`/`data-part` anatomy. The result is a catalog you can restyle
with plain CSS, design tokens, or Tailwind without forking a single component.
