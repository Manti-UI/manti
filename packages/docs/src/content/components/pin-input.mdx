# PinInput

A segmented code entry backed by the Zag.js pin-input machine. The machine owns
focus movement between cells, paste handling, and form wiring; you set the
length and character type.

<Demo name="pin-input/basic" center />

## Masked

Set `mask` to obscure the entered characters like a password field — useful for
PINs rather than one-time codes.
