# Drawer

An edge-anchored panel that slides in from a screen edge. It shares the Zag.js
dialog machine with [Dialog](/components/dialog) — the same focus trapping,
scroll locking, and dismissal — restyled to anchor to an edge and slide in.

<Demo name="drawer/basic" center />

## Basic

`trigger` is the element that opens the drawer — it's cloned with the machine's
trigger props. Pass `title`, `description`, and body content as `children`.
`footer` accepts a render function receiving `{ close }`, so action buttons can
dismiss the drawer.

<Demo name="drawer/basic" center />

## Placement

`placement` sets the edge the drawer slides in from: `right` (default), `left`,
`top`, or `bottom`. `size` controls the panel's extent along its axis — the
width for `left`/`right`, the height for `top`/`bottom`.
