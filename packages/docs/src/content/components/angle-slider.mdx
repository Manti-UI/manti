# AngleSlider

A circular angle picker backed by the Zag.js angle-slider machine. Drag the thumb
or use the arrow keys to choose an angle from 0 to 360 degrees.

<Demo name="angle-slider/basic" center />

## Basic

Set `defaultValue` for the starting angle and `step` for how finely it snaps.
Turn on `showValue` to print the current degree below the dial.

<Demo name="angle-slider/basic" center />

## Marks

Pass `marks` to render tick marks around the dial. Pair them with a matching
`step` so the thumb snaps to each mark.

<Demo name="angle-slider/marks" center />
