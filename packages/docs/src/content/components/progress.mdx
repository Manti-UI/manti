# Progress

A determinate or indeterminate progress indicator backed by the Zag.js progress
machine. It renders as a linear bar or a circular ring.

<Demo name="progress/basic" />

## Linear

The default `linear` variant fills a horizontal track. Set `showValue` to print
the formatted percentage next to the label.

<Demo name="progress/basic" />

## Circular

Set `variant="circular"` to render a ring, with the percentage centered inside
when `showValue` is on.

<Demo name="progress/circular" center />
