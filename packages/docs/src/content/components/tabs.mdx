# Tabs

Tabbed navigation backed by the Zag.js tabs machine. Pass an `items` array and
the machine owns roving focus, keyboard arrows, and the active indicator.

<Demo name="tabs/basic" />

## Basic

Each item is `{ value, label, content }` — `value` keys the tab, `label` is the
trigger, and `content` is the panel shown when it's active.

<Demo name="tabs/basic" />

## Variants

`variant` sets the visual style: `line` (a rail under the active tab), `pill` (a
translucent segmented control with a sliding thumb), or `soft` (a translucent pill
on the active tab, no enclosing track).
