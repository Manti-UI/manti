# HoverCard

A rich floating preview backed by the Zag.js hover-card machine. The machine owns
the open and close delays and the positioning; you supply the trigger and the
card content.

<Demo name="hover-card/basic" center />

## Basic

`trigger` is the element that reveals the card — it's cloned with the machine's
trigger props, so it works on hover and on keyboard focus. The card content is
passed as `children`.
