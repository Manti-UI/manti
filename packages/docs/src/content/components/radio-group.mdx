# RadioGroup

A set of mutually exclusive options, backed by the Zag.js radio-group machine.
Pass an `items` array and the machine owns roving focus, keyboard arrows, and
form wiring.

<Demo name="radio-group/basic" />

## Basic

Each item is `{ value, label, disabled? }` — `value` keys the option, `label` is
the text, and `disabled` greys it out.

<Demo name="radio-group/basic" />

## Horizontal

Set `orientation="horizontal"` to lay the options out in a row.
